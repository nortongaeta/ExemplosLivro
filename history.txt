git status
- mostra as alterações pendentes

git pull origin master
- atualizar a ultima versão do branch

Obs:
inicialmente a pasta do projeto foi criada como su.
Após isso os comandos git estavam requerendo sudo.
Para corrigir, coloquei o usuário como owner da pasta com o comando:

chown -R <usuário> <pasta do projeto>
